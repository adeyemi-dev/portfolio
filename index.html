Power Automate + Databricks Integration

Built an event-driven pipeline that connects SharePoint file uploads to Databricks table refreshes via Power Automate. Whenever a source file is modified or replaced in SharePoint, a trigger fires a Databricks job to ingest and process the updated data in near real-time. This eliminated manual intervention, reduced data latency dramatically, and ensured downstream reports always reflect the latest available data without any scheduled polling overhead.
